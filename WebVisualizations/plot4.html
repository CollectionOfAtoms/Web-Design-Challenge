Analysis: Latitude vs. Wind Speed

This data set looks pretty random to me if we just look at windspeeds below 20 mph. Which probably speaks to humans having a tendency to build cities in places that are not excessively windy to begin with. We can say that windspeeds look to be considerably higher in the deep global south, which I suppose makes sense. If winds tend to be oriented east-west, then we'd expect it to speed up near the poles.

There are also a number of outliers in the northern hemisphere. I suppose that it's possible that wind speeds correlate positively with warmer temperatures, but the data don't make it obvious here.
